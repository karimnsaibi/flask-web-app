Site Data Documentation

Reference guide for the data fields logged in the Site Management module.

Field Descriptions

Field Name Data Type Description Example
Region Text The administrative governorate where the site is located. Tunis, Ariana
Site Code Alphanumeric Unique identifier for the GSM site. 1234, T001
Delegation Text Specific municipality or local area within the region. La Marsa
Site Name Text Descriptive name of the site location. Marsa Plage
X / Y Float Geographic coordinates (Lambert or GPS) for mapping. 36.8, 10.1
HBA Float (Meters) Height Above Ground (Hauteur Bas de l'Antenne). 22.5
Supplier Text Manufacturer of the reliable radio equipment. Huawei, Ericsson
Access Text Transmission medium used (Fiber vs Microwave/FH). FO, FH
Antenna Text Type of antenna configuration. MIMO, Omni
Surface Text Location type (Rooftop vs Green field). Terrasse, Terrain nu
